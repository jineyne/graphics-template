# 그래픽 탬플릿

그래픽 개발을 시작할 수 있게 도와주는 프로젝트 입니다.

## 요구사항

 * Visual Studio 

## 설치 방법

 1. Command Prompt를 열어 아래 커맨드를 실행합니다
```batch
setup
```

 2. 이후 Visual Studio Prompt를 열어 아래 커맨드를 실행합니다.

 검색창에 x64 Native Tools 를 검색하면 찾을 수 있습니다.

 ```batch
build
 ```

 이후 Source 폴더에서 개발을 시작하면 됩니다
